# CoinHippo
This is the frontend of [coinmarketcoins](https://coinhippo.io).
https://www.coinmarketcoins.com/
### Prerequisites
- node >= 16.0.0
- [CoinHippo API](https://github.com/coinmarketcoins-Labs/coinmarketcoins-api)

## Run on [localhost:3000](http://localhost:3000)
```bash
yarn
yarn dev
```
